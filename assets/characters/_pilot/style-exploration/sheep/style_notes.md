# 羊 造形言語・画風探索メモ

## 目的

WORLDAWN の `ground_sheep`（羊 / Sheep、ground、normal、initial）を題材に、「実在生物ベースのスタイライズド・コレクション生物（Stylized Zoological Collectible Creature）」として成立し得る6種類の造形言語を可視化する。今回は完成案の決定ではなく、ユーザーが比較して次の方向を選べる状態を作ることを目的とする。

## 前回20枚の失敗原因

- 実在動物を細密に描くことが中心となり、自然誌イラストやストック素材に近づいた。
- 毛、羊毛、皺、鱗、針などの高周波テクスチャが、大きな形やシルエットより目立った。
- 候補差が造形言語ではなく、同じデザインのポーズ差分に留まった。
- ゲーム内で集めたくなる顔、比率、色面、玩具化しやすい塊の設計が弱かった。
- 背景処理と出力寸法が指定どおりにならず、生成側の自己採点もユーザーの好みを代替できなかった。

## STYLE 01（既生成・比較基準）

`style_01_rounded_friendly.png` は本継続作業より前に生成済みの比較基準であり、本作業では再生成・修正・上書きをしていない。元生成物と保存ファイルの SHA-256 はともに `e44a15904a1fe5df162d7bb9f0777ee241bae8308c2b8d78d89737564afa1858` で一致する。

長所：丸い体型、太い巻き角、大きな羊毛房、明快な輪郭により、写実から離れた親しみやすい収集物として読める。

弱点：羊毛が泡・雲・ポップコーンの集合に見えやすく、胸・胴・腰で房の流れに差が少ない。顔と目尻が汎用的な「かわいいゲーム動物」や人間的・化粧的表現に寄り、横長瞳孔、角の自然差、WORLDAWN 固有の記憶点が弱い。

| STYLE | 方向 | STYLE 01から変えた主な要素 |
|---|---|---|
| 01 | 丸みと親しみ | 短く丸い胴体、やや大きな頭部、短めの脚、太い角、丸い大房を使う比較基準。 |

| 03 | 上品さと洗練 | 細めで長い顔、自然な脚長、細い開放的な角、穏やかなまぶたと横長瞳孔、S字曲線で連続する層状の羊毛、繊細な輪郭へ変更。 |
| 04 | コンパクトな収集ゲーム型 | 大きな頭部、低重心、短い脚、太く単純な角、胸・胴・腰を少数の大きな羊毛ブロックへ圧縮し、最も強い外周輪郭を採用。 |

| 06 | かわいさと力強さの中間 | 中程度の頭部と胴体、角内側の大きな負の空間、頬から前脚へ流れる胸のクレセント、腰で向きが変わる中房により、親しみと強さを両立する記憶点を追加。 |

## 全案で固定した条件

- 成体の家畜雄羊1体。巻き角2本、耳2つ、脚4本、偶蹄、短い尾、顎髭・翼・余分な身体部位なし。
- 全身の立ち姿、身体は3/4方向、頭はこちらへ少し向け、4脚と4蹄を確認できる構図。
- 暖かいアイボリー／クリームの羊毛、淡いベージュの顔と脚、黄褐色の角、濃いブラウンの蹄と鼻、アンバー／ダークブラウンの瞳。
- 正方形、中央配置、ほぼ同じカメラ距離・視点・画面占有率、左上前方からの光。
- 明るいウォームグレー背景。地面、足元影、風景、緑、マゼンタ、市松模様なし。
- 大きな色面、2〜3段の陰影、ブラウン系の色付き輪郭。毛の一本描写、細かな角溝、装備、衣装、発光、文字なし。
- STYLE 01を無変更で保持するため、画像生成機能の正方形出力である1254×1254へ全6枚を統一した。

## 自動排除した画像

- STYLE 03 初回案：口角が人間的な微笑みに寄ったため排除し、閉じた中立口と横長瞳孔を明示して再生成した。
- STYLE 04 初回案と直前案：頭部・短脚・低重心が不足し、羊毛の分割と角の華やかさも normal のコンパクト型から外れたため排除した。ユーザー指示に従い、頭部を大きく、脚と重心を低くし、角を小さくしたうえで、羊毛を頭頂・胸・横長胴・後部の4ブロックだけへ再設計した。
- STYLE 05 初回案と中間案：初回は羊毛の縁と陰影が細かく自然誌・ストック寄りへ戻り、中間案は逆に全身が滑らかな樹脂フィギュア状になったため排除した。自然な雄羊の骨格を保ちつつ、胸・背・腹・腰の大領域へ少数の柔らかな不規則輪郭を与える平面的なゲーム造形へ再設計した。
- STYLE 06 初回案と中間案：初回は身体の向きが他案と左右逆で、中間案は01に近い丸房、過大な角、装飾布に見える長いS字の胸毛が指定方向から外れたため排除した。角内側のずれた涙形の余白と、顎周り・胸・胴・腰で切り替わる羊毛方向を記憶点として再設計した。

排除は明確な禁止事項・比較条件・指定方向との不一致だけを対象とし、魅力や好みの自己採点には使っていない。
