WORLDAWN キャラクター量産パック v3.0

1. Character_v3_mass_generation.xlsx をWorkへ添付してください。
2. assets/characters/_references/positive/ を、WORLDAWNリポジトリの同じ相対パスへ配置してください。
3. Workbookの「Work実行指示」シートにあるプロンプトをWorkへ貼り付けてください。
4. 最初は generation_queue の Batch=0（8体）だけを生成します。
5. 8体承認後に、Batch=1から20体ずつ量産します。

- 1体1生成。客観的な技術的不良時だけ最大1回修正。
- 12枚すべてを毎回参照せず、各行に割り当てた参照コードだけを使います。
- 正式データ、ID、rarity、アプリコード、既存正式画像は変更しません。
