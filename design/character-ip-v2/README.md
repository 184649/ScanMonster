# Character IP V2 design area

This directory contains Phase 2A non-production JSON Schemas and examples. Nothing here is imported by the application, server, catalog generators, seed, or production asset manifest.

- `sampleOnly: true` is required in example documents.
- `prototype_character_*` and `prototype_form_*` are disposable design identifiers, not production IDs.
- Legacy IDs are references and are never redefined.
- Passing schema checks does not approve a character or a migration.
