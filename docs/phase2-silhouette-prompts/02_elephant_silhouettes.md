# Gate 1 prompt — Elephant

```text
WORLDAWNの新しいIPキャラクター設計を検証するため、ゾウを主要生物モチーフにした白黒シルエット案を1枚のボードへ8案生成してください。完成イラスト、既存画像の修正、既存キャラクターの模倣ではありません。参照画像は使用せず、特定の既存IPのキャラクター、顔、身体比率、外周、ポーズを再現しないでください。

出力は4列×2行。上段左から右、次に下段左から右の8位置を、生成後にA〜Hとして扱います。画像内にはA〜Hを含む文字、番号、記号、説明を書かないでください。各案の全身が同じ縮尺で見え、鼻、耳、四肢など主要部位が切れず、案と案の間に十分な白い余白がある構図にしてください。

各案は同じキャラクターのポーズ差分ではなく、独立した8つのキャラクター企画です。ゾウとして読める範囲で、頭部と胴体の比率、重心、脚の長さ、鼻・耳・頭部・肩・四本の支持脚など種固有部位のまとめ方、外周シルエット、姿勢、玩具化した際の主要ブロック構造を明確に変えてください。筋肉、装甲、写実的な皮膚、耳の大きさだけへ逃げず、道を確かめてから一歩を決める、穏やかで観察力のある大きな案内役の性格を姿勢だけから想像できる案にしてください。

各案の主要な記憶点は必ず一つだけにしてください。候補となる設計軸は、低く前へ伸びる鼻と高い頭肩の塊がつくる橋、前重心と一歩目の支持、耳と胴体の負の空間などです。ただし、一案へ複数のフックを詰め込まず、8案が同じ鼻形状の微差にもならないようにしてください。64pxへ縮小しても外周だけで区別でき、3〜9個程度の大きな部品へ単純化できる形を優先してください。

描画は黒一色の完全に塗りつぶしたキャラクター形状と、白一色の背景だけに限定してください。目、瞳、口、鼻孔、表情の描き込み、顔の内部線、皮膚、皺、毛、質感、模様、色、グラデーション、陰影、ハイライト、輪郭線内のディテールを描かないでください。地面、台座、足元影、背景、風景、演出、発光、装備、衣服、鞍、冠、持ち物、文字を加えないでください。市松模様や透明背景表現も不要です。

8案を一枚だけ生成した後は、良し悪しの採点、ランキング、推奨、最終案の自動選定、追加生成を行わず停止してください。ユーザーがA〜Hから完成形を見たい案を最大2つ選ぶまで次の工程へ進まないでください。どれも選べない場合は「全部違う」という判断を受け入れてください。
```
