# Gate 1 prompt — Snake

```text
WORLDAWNの新しいIPキャラクター設計を検証するため、ヘビを主要生物モチーフにした白黒シルエット案を1枚のボードへ8案生成してください。完成イラスト、既存画像の修正、既存キャラクターの模倣ではありません。参照画像は使用せず、特定の既存IPのキャラクター、顔、身体比率、外周、ポーズを再現しないでください。

出力は4列×2行。上段左から右、次に下段左から右の8位置を、生成後にA〜Hとして扱います。画像内にはA〜Hを含む文字、番号、記号、説明を書かないでください。各案の全身が同じ縮尺で見え、頭から尾端まで主要部位が切れず、案と案の間に十分な白い余白がある構図にしてください。

各案は同じキャラクターのポーズ差分ではなく、独立した8つのキャラクター企画です。ヘビとして読める範囲で、頭部と胴体の比率、重心、全身の長さと太さの配分、頭部・顎・首への移行・胴の巻き・尾など種固有部位のまとめ方、外周シルエット、姿勢、玩具化した際の主要ブロック構造を明確に変えてください。単なるS字の微差、コブラの威嚇、魔法の使い魔へ寄せず、低い安定した巻きの大部分で地面の振動を聞き、一部だけを短く起こす、静かで安心感のある聞き手の性格を姿勢だけから想像できる案にしてください。

各案の主要な記憶点は必ず一つだけにしてください。候補となる設計軸は、非対称の低い巻きと短い首の間にできる守られた負の空間、頭部の楔形と巻きの対比、接地する大きな塊と一点だけ起きる方向性などです。ただし、一案へ複数のフックを詰め込まず、8案が同じ巻き方の微差にもならないようにしてください。64pxへ縮小しても外周だけで区別でき、3〜6個程度の大きな部品へ単純化でき、自立するフィギュアへ変換可能な形を優先してください。

描画は黒一色の完全に塗りつぶしたキャラクター形状と、白一色の背景だけに限定してください。目、瞳、口、舌、牙、鼻孔、表情の描き込み、顔の内部線、鱗、質感、模様、色、グラデーション、陰影、ハイライト、輪郭線内のディテールを描かないでください。地面、台座、足元影、背景、風景、演出、発光、装備、衣服、宝石、首飾り、武器、持ち物、文字を加えないでください。市松模様や透明背景表現も不要です。

8案を一枚だけ生成した後は、良し悪しの採点、ランキング、推奨、最終案の自動選定、追加生成を行わず停止してください。ユーザーがA〜Hから完成形を見たい案を最大2つ選ぶまで次の工程へ進まないでください。どれも選べない場合は「全部違う」という判断を受け入れてください。
```
