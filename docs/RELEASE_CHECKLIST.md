# WORLDAWN リリース前チェックリスト

## 基本

- [ ] `package.json` の name が `worldawn`
- [ ] `app.json` の name/slug が WORLDAWN 用
- [ ] ユーザー向け画面から ScanMonster 表記が消えている
- [ ] Expo Goで起動できる
- [ ] iPhone実機で初期画面まで表示される

## 初回リリース機能

- [ ] 初回カテゴリ選択画面が表示される
- [ ] land / water / sky / bug から1つ選べる
- [ ] 選択後、選んだカテゴリだけが解放済みになる
- [ ] 2回目以降は初回カテゴリ選択が表示されない
- [ ] ホームにDP、解放済みカテゴリ数、次の解放コストが表示される
- [ ] DPでカテゴリを解放できる
- [ ] 解放コストが「何個目か」で決まる
- [ ] DP不足時に不足DPが分かる
- [ ] 気配ブーストを解放済みカテゴリにだけ使える
- [ ] 気配ブーストは300DP消費する
- [ ] 気配ブーストは次の10回の有効スキャンで減る
- [ ] 同日同コードブロックでは気配ブーストが減らない
- [ ] 気配ブーストでレア確率が上がらない
- [ ] 称号が条件達成で解放される
- [ ] 称号をマイページ表示用に設定できる

## スキャン

- [ ] カメラ権限を要求できる
- [ ] JAN/EAN/UPC/Code128/Code39を読み取れる
- [ ] QRコードを読み取れる
- [ ] 写真ライブラリから画像内コードを読み取れる
- [ ] 複数コードを処理できる
- [ ] 3秒以内の同一コードは重複読取として無視される
- [ ] 同じコードの同日2回目は抽選されない
- [ ] 同じコードの同日2回目はDP付与なし
- [ ] 有効スキャンでは解放済みカテゴリから抽選される
- [ ] 未解放カテゴリのキャラが出ない
- [ ] 未発見優先になっていない
- [ ] 所持済みキャラも候補に含まれる

## レア

- [ ] レア確率が1〜3%の範囲
- [ ] DPでレア確率が上がらない
- [ ] 気配ブーストでレア確率が上がらない
- [ ] レアは解放済みカテゴリに紐づくものだけ出る

## 図鑑・詳細

- [ ] 図鑑が表示される
- [ ] 発見済み/未発見が分かる
- [ ] キャラカードから詳細へ遷移できる
- [ ] キャラ詳細で画像が表示される
- [ ] 画像がない場合もfallbackで落ちない
- [ ] 初発見日、最終発見日、発見回数が表示される
- [ ] お気に入りが動く
- [ ] 個体値、個体差、フォーム、背景、フレーム導線が出ていない

## 対象外機能

- [ ] 探索タブが表示されていない
- [ ] ミッションタブが表示されていない
- [ ] 研究導線が主要画面に出ていない
- [ ] カテゴリ図鑑導線が主要画面に出ていない
- [ ] フレンド、交換、ランキング、課金、広告が主要導線に出ていない

## プライバシー

- [ ] `rawBarcode` / `rawQrValue` / `normalizedValue` を永続保存していない
- [ ] 保存は `sourceHash` 中心
- [ ] 同日同コード制限は `sourceHash + localDate`
- [ ] 正確な位置情報を保存しない
- [ ] 位置情報を出現制御に使わない
- [ ] 共有内容にバーコード数字、QR内容、sourceHash、正確な時刻、位置情報が含まれない

- [ ] Android package が確定している
- [ ] privacyPolicyUrl が本番URL
