## 2. Project summary

> WORLDAWNは、戦闘・捕獲・育成を中心とするゲームではなく、「出会う、発見する、記録する、もう一度会う」体験です。ユーザーはコードを読み取り、キャラクターを発見し、キャラクターごとの公式発見番号と発見証明を受け取り、図鑑・履歴・カレンダーへ残します。既存の有名IPを具体的に模倣せず、発見したくなり、名前を覚え、一体ごとに好きになる理由がある独立した長期IPを目指します。
