# Phase 1 実在生物図鑑イラスト：アートバイブル＆マスタープロンプト

WORLDAWN の **normal / rare** は、人格付きオリジナルキャラクターではなく、**実在生物のプレミアム図鑑イラスト**として描く。
本書は画風の正本であり、ChatGPT Work へそのまま貼り付けて使う**マスタープロンプト**を含む。

- 対象：normal（本Phaseのパイロットは初期 ground normal の5種）
- 対象外：legendary / secret（WORLDAWN独自キャラクターとして別設計）／案内キャラクター
- 画像生成は **Phase 1B**。本書は生成前の設計・基準・プロンプトのみ。

---

## 1. 画風の正式名称

**プレミアム生物コレクションイラスト**（内部表現：**Premium Zoological Collectible Illustration**）

写真的な完全写実でも、人格付きマスコットでもない。**実在生物として正確でありながら、収集したくなる商品性を持つ**中間点。

### 同時に満たすこと
- 一目で元の生物が判別でき、**近縁種と区別できる**
- 骨格・脚数・関節・耳・尾・角・鼻・口・模様が**生物学的に不自然でない**
- 実在生物としての**生命感**がある／写真の切り抜きに見えない
- 学校図鑑の挿絵より**華がある**／高品質モバイル収集ゲームの収集物に見える
- **10〜30代が集めたくなる**／子ども向けの幼いマスコットに見えない
- 過剰に怖い・生々しい・グロテスクにしない
- **小さなスマホ画面でも識別できる**／一覧に並べてもシルエットが潰れない
- SNS共有カードに置いて見栄えがする／背景なしで1枚の作品として成立する
- **normal でも地味に見えない**が、**rare / legendary の特別感を奪うほど派手にしない**

## 2. 描画スタイル（全種共通・文章だけで再現可能なレベル）

| 要素 | 定義 |
|---|---|
| 全体 | 高品質なモバイルゲーム用生物イラスト。日本・韓国の上質なコレクションゲームアートに通じる洗練 |
| 立ち位置 | **アニメ調とナチュラルな生物画の中間**。セルシェーディングと柔らかな厚塗りの中間 |
| 輪郭 | 明快だが、**均一で太い真っ黒な主線にしない**。輪郭線は**素材と光に合わせて色味を変える**（暗部は濃く、光を受ける縁は明るく） |
| 陰影 | 基本 **2〜3段**。主要な明暗が**サムネイルでも読める**こと |
| 密度 | **顔と種固有部位を最も丁寧に**描く。それ以外は情報量を落として視線を誘導する |
| 素材 | 毛・短毛・羊毛・皮膚・鱗・角・爪・針を**描き分ける**（同じ塗りで済ませない） |
| 配色 | 鮮やかだが**実在生物から逸脱しない**。彩度・明度・コントラストは自然色の範囲内で調整 |
| 光 | 自然な反射光を加え、**透明背景でも立体感**を出す。片側からの主光＋弱い環境光 |
| 禁止 | 過剰なエフェクト／魔法／発光／オーラ／台座／地面／草／岩／風景／**足元の不透明な楕円影** |
| 出力 | **完全透明背景・全身・単体・中央配置・主要部位を切らない・1024×1024・PNG** |

## 3. キャラクター化の範囲（normal）

**人格付きキャラクターにしない。ただし無機質な標本にもしない。**

**許可**：生物らしい自然な視線／警戒・好奇心・落ち着きなどの自然な状態／種固有の自然な姿勢／歩く・伏せる・振り向く・見上げるなどの自然な動作／特徴部位が読みやすくなる**軽度の**比率整理／小画面向けのシルエット整理／自然色の範囲内での彩度・明度・コントラスト調整

**禁止**：人間のような笑顔・眉・感情演技／頬に手を当てる・胸に手を置く・手を振る／人間のような直立／衣服・武器・防具・帽子・マフラー・バッグ・ペンダント・宝石・ゴーグル／職業設定・冒険者設定・魔法／本来存在しない角・翼・耳・尾／巨大なアニメ目／大頭短足の赤ちゃん化／ぬいぐるみ化／デフォルメだけでかわいさを作る／既存の有名IPキャラクターへの類似

---

## 4. 構図（全種を同じポーズにしない／画風と品質は統一）

共通条件：単体・全身・身体が画面内に収まる・顔が確認できる・**正面固定ではない**・極端な俯瞰/煽りではない・身体の重なりで主要部位を隠さない・複雑なアクロバットをしない・小アイコンでもシルエットが読める・**左右反転しても破綻しない**・正方形に自然に収まる

種ごとに最適なものを選ぶ：立つ／伏せる／座る／歩き始める／振り向く／見上げる／低い姿勢で観察する／種固有の自然な姿勢

---

## 5. 生成サイクル（1生物あたり）

1. **4案を内部設計**（ポーズ・重心・顔の向き・特徴部位の見せ方・軽度の色調・シルエットを変える。**別種・別個体・別画風にはしない**）
2. **100点満点で自己評価**
3. **最高点 88点未満なら失敗原因を分類し、構図または設計から再設計**
4. **最大3巡**
5. **最終的に1枚だけ出力**

### 採点（合計100点）
| 項目 | 配点 |
|---|---|
| 種の識別性 | 20 |
| 生物学的自然さ | 20 |
| プレミアムゲームアートとしての魅力 | 15 |
| シルエットの読みやすさ | 10 |
| 素材表現 | 10 |
| 小画面での認識性 | 10 |
| 全5種での画風統一 | 10 |
| 独自性・既存IP非類似 | 5 |

### 自動不合格（点数にかかわらず不合格）
脚・指・角・尾・耳などの数や構造が不自然／身体部位の融合／本来ない身体部位／人間化／マスコット化／巨大なアニメ目／衣装・装備／背景や地面／**透明風の市松模様**／単色背景の残存／主要部位の切れ／既存有名IPへの過度な類似／写真の切り抜きにしか見えない／普通の教材用図鑑イラストにしか見えない

---

## 6. 5種全体での統一評価（個別に良いだけでは合格にしない）

5種を**並べた状態**で確認する：同じゲームの収集物に見える／同じ照明設計／同じレンダリング品質／輪郭線の強さが大きくばらつかない／彩度とコントラストが大きくばらつかない／写真・写実画・アニメ絵が混在していない／全員が同じポーズでない／全員が同じ顔の向きでない／全員が同じかわいさに寄っていない／大型と小型の存在感が適切／毛・皮膚・鱗・針の素材差が明確／一覧で各生物を区別できる／normal として統一されている／legendary のような過剰な派手さがない

**5種のうち1種だけ画風が外れている場合は、その1種だけを再生成する。**

## 7. ChatGPT Work 用マスタープロンプト（一括コピー）

以下を**そのまま1つのブロックとしてコピー**し、`【差し替え】` の箇所を各生物の個別設計データ（`docs/legacy-character-prompts/phase1-pilot-prompts/` の各ファイル）で置き換えて使用する。**参考画像は不要**。

```text
あなたはモバイル収集ゲーム「WORLDAWN」の生物図鑑イラストを制作する、シニア生物イラストレーター兼アートディレクターです。
これから「1体の実在生物」のプレミアム図鑑イラストを1枚だけ制作します。

■ 制作対象
和名: 【差し替え：和名】
英名: 【差し替え：英名】
分類: 【差し替え：分類】
必ず残す識別特徴（優先順位順）:
【差し替え：識別特徴を3〜6個、優先順位順で記載】
近縁種との差（混同を防ぐ）:
【差し替え：近縁種名と、それと区別するための決定的な違い】
体表素材:
【差し替え：毛/短毛/羊毛/皮膚/鱗/針など、質感の具体的記述】
自然な体色:
【差し替え：実在する体色。多くとも3〜4色の範囲】
推奨構図:
【差し替え：この種の魅力と認識性が最も高くなる自然な姿勢】
避ける構図:
【差し替え：この種で不自然・誤読されやすい構図】
誤生成しやすい部位（特に注意）:
【差し替え：指の数、角の形、脚の関節、尾など】

■ 画風（絶対条件）
「プレミアム生物コレクションイラスト」= 写真的な完全写実ではなく、人格付きマスコットでもない、その中間。
- 高品質なモバイルゲーム用の生物イラスト。日本・韓国の上質なコレクションゲームアートに通じる洗練。
- アニメ調とナチュラルな生物画の中間。セルシェーディングと柔らかな厚塗りの中間。
- 輪郭は明快だが、均一で太い真っ黒な主線にはしない。輪郭線は素材と光に合わせて色味を変える。
- 陰影は基本2〜3段。主要な明暗が小さなサムネイルでも読めること。
- 顔と種固有部位を最も丁寧に描き、他は情報量を落として視線を誘導する。
- 毛・短毛・羊毛・皮膚・鱗・角・爪・針を明確に描き分ける。
- 色は鮮やかだが実在生物から逸脱しない。自然な反射光を加え、透明背景でも立体感を出す。
- 学校図鑑の挿絵より華があり、高品質な収集ゲームの収集物に見えること。10〜30代が集めたくなる質感。
- normalとして統一感を保ち、伝説級の派手さ（発光・オーラ・エフェクト）は使わない。

■ 生物としての扱い（重要）
実在生物として正確に描く。人格付きキャラクターにはしない。ただし無機質な標本にもしない。
- 許可: 生物らしい自然な視線、警戒・好奇心・落ち着きなどの自然な状態、種固有の自然な姿勢と動作、
  特徴部位が読みやすくなる軽度の比率整理、小画面で読みやすくするためのシルエット整理。
- 禁止: 人間のような笑顔・眉・感情演技、手を振る、頬に手を当てる、人間のような直立、
  衣服・武器・防具・帽子・マフラー・バッグ・ペンダント・宝石・ゴーグル、職業や冒険者の設定、魔法、
  本来存在しない角・翼・耳・尾、巨大なアニメ目、大頭短足の赤ちゃん化、ぬいぐるみ化、
  デフォルメだけでかわいさを作ること、既存の有名IPキャラクターに似せること。

■ 制作手順（必ずこの順で内部実行すること）
ステップ1: 上記の生物について、ポーズ・重心・顔の向き・特徴部位の見せ方・軽度の色調・シルエットが
  互いに異なる「4案」を内部で設計する。ただし4案とも同一の種・同一の個体・同一の画風であること
  （別種や別デザインに変えてはいけない）。
ステップ2: 4案それぞれを次の配点で100点満点で自己採点する。
  種の識別性20点 / 生物学的自然さ20点 / プレミアムゲームアートとしての魅力15点 /
  シルエットの読みやすさ10点 / 素材表現10点 / 小画面での認識性10点 / 画風統一10点 / 独自性・既存IP非類似5点
ステップ3: 次のいずれかに該当する案は、点数にかかわらず不合格として除外する。
  脚・指・角・尾・耳などの数や構造が不自然 / 身体部位の融合 / 本来ない身体部位 / 人間化 / マスコット化 /
  巨大なアニメ目 / 衣装や装備 / 背景や地面が描かれている / 透明風の市松模様 / 単色背景が残っている /
  主要部位が切れている / 既存の有名IPに似すぎている / 写真の切り抜きにしか見えない /
  普通の教材用図鑑イラストにしか見えない。
ステップ4: 最高得点が88点未満の場合は、失敗原因を分類したうえで構図または設計から作り直し、
  ステップ1へ戻る。この再設計は最大3巡まで行う。
ステップ5: 88点以上の最良案を1つだけ選び、その1枚だけを最終画像として出力する。
  途中の候補画像や比較画像は出力しない。

■ 出力仕様（厳守）
- 1024×1024ピクセル、正方形、PNG。
- 完全な透明背景（アルファチャンネルあり）。背景・地面・草・岩・風景・台座を描かない。
- 足元に不透明な楕円影を置かない。透明を表す市松模様を描かない。単色背景を残さない。
- 生物は1体のみ。全身。中央配置。主要部位（頭・四肢・尾・角など）を画面外で切らない。
- 文字・ロゴ・UI・フレーム・枠線・キャプション・透かし・署名を入れない。
- 左右反転しても破綻しない構図にする。小さなアイコンに縮小してもシルエットで種が判別できること。
- 既存の有名IPやキャラクターを模倣しない。完全にオリジナルの作画とする。

最終出力は、上記を満たした画像1枚のみとします。
```

## 9. 生成後の流れ

1. 生成画像を `assets/characters/_pilot/phase1/<character-id>/candidates/` に置く
2. `assets/characters/_pilot/phase1/<character-id>/reports/evaluation.json` に視覚評価を記録
3. 88点以上かつ automaticReject=false の案を `selected/` へ置き、`assets/characters/phase1-pilot.json` の `selectedCandidate` に指定
4. `npm run validate:phase1-pilot` で技術検証
5. 正式採用は `docs/PHASE1_ASSET_PROMOTION.md` の昇格手順に従う（**本Phaseでは実施しない**）
