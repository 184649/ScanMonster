# LEGEND — 絶滅した実在生物

対象：恐竜、古生代生物、絶滅哺乳類・鳥類・海洋生物など、**すでに絶滅した実在の生きもの**。共通方針は [00_ART_DIRECTION.md](00_ART_DIRECTION.md)。

保存先：`assets/encyclopedia/legend/<id>.png`

## LEGEND の定義（誤用を防ぐ）

- **現生生物をLEGENDにしない。** 強い動物・大型動物という理由で分類しない。
- **神話・伝承・空想上の生きものはLEGENDではなくSECRET。**
- 復元は**現在の研究と化石証拠**にもとづく。古い図鑑由来の復元を無批判に踏襲しない。
- **復元に議論がある部分をデータとして保持し、画像の注記へ回す。** 不明部分を派手な装飾で埋めない。

## 画像生成プロンプト（そのままコピー）

```text
あなたは古生物学的復元を重視するWORLDAWN自然史図鑑の復元画家です。
すでに絶滅した実在の生きものを、科学的に妥当な復元画として1枚だけ制作します。

■ 対象
和名: {japaneseName}
英名: {englishName}
学名: {scientificName}
分類: {taxonomy}
生息年代: {geologicalPeriod}
化石発見地域: {fossilLocations}
推定サイズ: {estimatedSize}

■ 化石証拠から確実性が高い特徴（必ず正確に描く）
{wellSupportedFeatures}

■ 復元に議論がある特徴（推測で断定しない）
{uncertainFeatures}
これらの部位は、現在の研究で広く支持されている最も保守的な解釈で描いてください。
議論がある部分を、独自の解釈や派手な装飾で埋めてはいけません。

■ 復元の方針
現在の研究と化石証拠にもとづく、科学的に妥当な姿で復元します。
昔の図鑑に由来する古い復元（引きずる尾、直立姿勢、鱗だけの体表など、現在は否定されているもの）を踏襲しません。
映画的な怪物化をしません。牙や爪を誇張せず、威嚇的な演出を加えません。
生きていた動物として自然な姿勢と重心にします。
軟部組織が化石に残らない部位は、近縁の現生群から妥当に推定した範囲にとどめます。

■ 禁止
証拠のない角、羽毛、皮膜、模様を足すこと。
不明部分を派手な装飾で補うこと。
過度なデフォルメ、巨大な丸目、人間的な表情。
架空の器官、宝石、鎧、装備、発光。
王冠やエフェクトなど、希少さを画像へ焼き込む表現。
文字、ロゴ、UI、フレーム、キャプション、透かし、署名。
地面、台座、影、風景、比較線、説明パネル、スケールバー。
既存の有名IPキャラクターや映画に登場する復元に似せること。

■ 画風
写真の複製ではなく、高品質な2Dの自然史復元画にします。
明快な輪郭、整理された色面、柔らかな多段階の陰影。
体表（鱗・羽毛・皮膚・甲殻）は、証拠が支持する範囲で質感を描き分けます。
現生生物の図鑑イラストと並べて画風が揃うこと。

■ 出力仕様（厳守）
1024×1024ピクセル、正方形、PNG。
本物の透明背景（アルファチャンネルを持ち、実際に完全透明な画素が存在すること）。
単色背景を残さない。透明を表す市松模様を描かない。足元に不透明な楕円影を置かない。
生きものは1体のみ。全身。中央配置。識別しやすい自然な3/4方向。
主要部位を画面外で切らない。
種名・学名・年代表記は入れない（後工程でUIが配置する）。

最終出力は、上記を満たした画像1枚のみとします。
```

## 生成後

1. `npm run validate:encyclopedia` を実行し、終了コード0を確認する。
2. 目視で確認する：古い復元になっていないか／証拠のない部位が足されていないか／怪物化していないか。
3. 図鑑UIでは `uncertainFeatures` を「議論がある復元」として表示する。**確定情報として断定表示しない。**
