# SECRET — 神話・伝承・空想・SF上の生物

対象：ドラゴン、フェニックス、ユニコーン、グリフォン、クラーケン、エイリアン、宇宙生命、WORLDAWN固有の空想生物。共通方針は [00_ART_DIRECTION.md](00_ART_DIRECTION.md)。

保存先：`assets/encyclopedia/secret/<id>.png`

## SECRET の扱い（最重要）

- **実在生物・絶滅生物と科学的事実として混在させない。** 図鑑UIでは見出しを「創作図鑑」とし、`fictionDisclaimer` を必ず表示する。
- 実在を示す科学的証拠はない旨を明示する。学名・保全状況・分布などの**科学フィールドを空想生物に付けない**。
- 既存作品のキャラクターを模倣しない。
- **発見方法は変更しない。** 現在ワールド完成条件で解放される生きものは、図鑑分類がSECRETになっても解放条件はそのまま。
- **未解放のSECRETは、存在・枠・件数・シルエット・ヒントを一切表示しない**（従来の完全秘匿仕様を維持）。

## 差し込む値

| プレースホルダ | 取得元（species-profiles.json） |
|---|---|
| `{name}` | `speciesJa` / `speciesEn` |
| `{fictionalOrigin}` | `fictionalOrigin`（由来） |
| `{mythologyOrWorldLore}` | `mythologyOrWorldLore`（神話・伝承） |
| `{fictionalEcology}` | `fictionalEcology`（WORLDAWN内の設定） |
| `{anatomyRules}` | 身体構造の一貫したルール |
| `{negativeRules}` | 個体ごとの禁止事項 |

## 画像生成プロンプト（そのままコピー）

```text
あなたはWORLDAWNの空想生物図鑑を担当するコンセプトアーティストです。
神話・伝承・空想上の生きものを、実在生物の自然史図鑑と並べても品質が揃うリアルなイラストとして1枚だけ制作します。

■ 対象
名称: {name}
由来: {fictionalOrigin}
神話・伝承上の位置づけ: {mythologyOrWorldLore}
WORLDAWN内での生態: {fictionalEcology}
身体構造のルール: {anatomyRules}

■ 制作方針
これは空想上の存在ですが、絵としては「生きている動物」に見えなければなりません。
身体構造に一貫した生物学的ロジックを持たせます。
関節の向き、体重を支える四肢、呼吸や採餌が成立する頭部、飛ぶなら翼面積と体重の釣り合いを考えます。
実在の動物から骨格・筋肉・体表の描き方を借り、説得力のある解剖にします。
質感（毛・羽・鱗・皮膚・甲殻）は実在生物と同じ精度で描き分けます。
自然史図鑑のページに並べたとき、画風と情報密度が揃っていること。

■ 禁止
既存の有名IPやゲーム・映画のキャラクターを模倣すること。
生物学的に破綻した関節、脚数、翼の付き方。
設定と無関係な装飾。宝石、鎧、装備、紋章、浮遊するパーツ。
過度なデフォルメ、巨大な丸目、人間的な笑顔・眉・まつ毛。
実在の生きものとして誤認させる描写（実在種そっくりに描いて空想要素だけ足す等）。
文字、ロゴ、UI、フレーム、キャプション、透かし、署名。
地面、台座、影、風景、説明パネル。
{negativeRules}

■ 出力仕様（厳守）
1024×1024ピクセル、正方形、PNG。
本物の透明背景（アルファチャンネルを持ち、実際に完全透明な画素が存在すること）。
単色背景を残さない。透明を表す市松模様を描かない。足元に不透明な楕円影を置かない。
生きものは1体のみ。全身。中央配置。自然で識別しやすい3/4方向。
主要部位を画面外で切らない。
名称や設定文は入れない（後工程でUIが配置する）。

最終出力は、上記を満たした画像1枚のみとします。
```

## 生成後

1. `npm run validate:encyclopedia` を実行し、終了コード0を確認する。
2. 目視で確認する：既存IPに似ていないか／関節・脚数が破綻していないか／実在種と誤認させないか。
3. 図鑑UIで `fictionDisclaimer` が表示されることを確認する。**科学的事実として断定表示していないこと。**
