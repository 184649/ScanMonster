# 虫・節足動物の特別指示（全分類に優先して適用）

対象：`bug` ワールドの全種、および他ワールドの節足動物・軟体動物など、**リアルに描くと敬遠されやすい生きもの**。

共通方針は [00_ART_DIRECTION.md](00_ART_DIRECTION.md)。分類テンプレート（NORMAL/RARE/LEGEND/SECRET）と**併用**し、衝突する場合は**この文書を優先**する。

## なぜ特別扱いが必要か

虫はリアルに描くほど「気持ち悪い」に寄り、一覧で並べたときに開きたくない画面になる。
図鑑としての正確さは保ったまま、**「美しい・かっこいい」側へ寄せる**必要がある。

collection として成立させるには、**正確さ ＞ 細密さ**。細部を足すほど敬遠される。

## 優先して描くもの

種の同定はここで担保する。

1. **シルエット** — 縮小しても種が分かる外形
2. **角・大顎・翅の形** — その種の記憶点になる構造
3. **体色と模様** — 色面として整理して描く
4. **体節の区切り** — 頭部・胸部・腹部の関係
5. **脚の本数と付き方** — 6本（昆虫）／8本（クモ類）を正確に

## 強調しすぎないもの（描くが、主役にしない）

「描かない」ではなく「**目立たせない**」。省略すると別種に見えるため、存在は保つ。

| 部位 | 扱い |
|---|---|
| 口器（大顎・口吻） | 形は正しく。牙のように誇張しない。開いて見せない |
| 複眼 | 面として描く。個眼の粒々を描き込まない |
| 毛・剛毛 | 生えている事実は残す。1本ずつ描かない。密集させない |
| トゲ | 輪郭として残す。鋭さを強調しない |
| 質感 | マットに。**濡れた・ぬめった・テカった質感にしない** |
| 関節の隙間 | 目立たせない。可動部の生々しい描写を避ける |

## 禁止

- 標本写真のような見え方（ピン刺し、展翅、背景紙、真俯瞰）
- ホラー感、威嚇姿勢、攻撃的な構図
- 極端なマクロ寄り（顔面のアップ、口器のアップ）
- 濡れた質感、粘液、体液
- 群れ・多数個体（1体のみ）
- 幼虫・蛹の生々しい描写（指定がある場合を除く）
- 人の手や指との対比（サイズ比較は文章で行う）

## 推奨する見せ方

- **やや上からの3/4方向**。真俯瞰（標本的）にも、真横（平面的）にもしない
- 翅は**閉じ気味〜半開き**。全開の展翅は標本に見える
- 触角は自然に前方へ。左右対称に張らない
- 脚は接地している姿勢。宙に浮かせない
- 色は彩度を保ちつつ、**光沢はハイライト1〜2箇所まで**

## プロンプトへの追記文（分類テンプレートの「■ 禁止」へ足す）

該当種を生成するときは、選んだ分類テンプレート（01〜04）の禁止事項に、次をそのまま追記する。

```text
■ 虫・節足動物としての追加指示（他の指示より優先）
種の同定は、シルエット、角や大顎や翅の形、体色と模様、体節の区切り、脚の本数で行います。
以下は「描かない」のではなく「目立たせない」でください。省略すると別種に見えるため、存在は保ちます。
- 口器（大顎・口吻）は形を正確に描きますが、牙のように誇張せず、開いて見せません。
- 複眼は面として描き、個眼の粒々を描き込みません。
- 毛や剛毛は生えている事実を残しつつ、1本ずつ描かず、密集させません。
- トゲは輪郭として残し、鋭さを強調しません。
- 体表はマットにします。濡れた質感、ぬめり、粘液、体液、強いテカリを描きません。
- 関節の隙間や可動部を生々しく描きません。

構図は、やや上からの3/4方向にします。真俯瞰にすると標本写真に見え、真横にすると平面的になります。
翅は閉じ気味から半開きにします。全開に展翅した姿は標本に見えるため避けます。
触角は自然に前方へ向け、左右対称に張りません。脚は接地させ、宙に浮かせません。
光沢はハイライト1〜2箇所までに抑えます。

次を禁止します。
標本写真のような見え方（ピン刺し、展翅、背景紙、真俯瞰）。
ホラー感、威嚇姿勢、攻撃的な構図。
顔面や口器の極端なマクロ寄り。
濡れた質感、粘液、体液。
複数個体。幼虫や蛹の生々しい描写。人の手や指との対比。

「気持ち悪い」ではなく「美しい・かっこいい」と感じられることを、細密さより優先してください。
```

## 生成後の確認

技術検査（`npm run validate:encyclopedia`）に加えて、**目視で**次を確認する。

- 一覧サムネイル相当（約120px）に縮小しても種が分かるか
- 縮小しても不快に感じないか
- 他ワールドの哺乳類・鳥類と並べて画風が揃っているか
- 標本写真・ホラーに見えていないか
- 口器・複眼・毛が主役になっていないか

**縮小表示で確認すること。** 等倍で良く見えても、一覧で不快なら不合格。
