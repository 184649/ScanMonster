# NORMAL — 現生生物の通常個体・通常形態

対象：現在も生きている生きものの、通常の形態。共通方針は [00_ART_DIRECTION.md](00_ART_DIRECTION.md)。

保存先：`assets/encyclopedia/normal/<id>.png`

## 差し込む値

`assets/characters/species-profiles.json` の**確認済み（reviewStatus: confirmed）**データから取る。未調査の項目を推測で埋めない。

| プレースホルダ | 取得元 |
|---|---|
| `{japaneseName}` / `{englishName}` | `speciesJa` / `speciesEn` |
| `{scientificName}` | `scientificName` |
| `{taxonomy}` | `taxonomy` |
| `{diagnosticFeatures}` | `diagnosticFeatures` |
| `{habitat}` | `habitat` |
| `{behavior}` | `behavior`（自然な姿勢・動作の根拠に使う） |
| `{speciesSpecificNegative}` | 種ごとの誤生成しやすい部位。近縁種との混同を防ぐ記述 |

## 画像生成プロンプト（そのままコピー）

```text
あなたはWORLDAWN自然史図鑑の専門イラストレーターです。
実在する生きもの1種を、科学的に正確な自然史イラストとして1枚だけ制作します。

■ 対象
和名: {japaneseName}
英名: {englishName}
学名: {scientificName}
分類: {taxonomy}
これは現生生物の通常個体・通常形態です。特殊な色彩変異や希少形態ではありません。

■ 最優先で正確に描くこと
識別特徴: {diagnosticFeatures}
生息環境から想定される自然な姿勢と体つき: {habitat}
観察される代表的な行動: {behavior}

対象種を一目で識別できることを最優先します。
骨格、体型、頭部の形、脚の数、指の数、翼・鰭・触角・体節の構造を正確に描きます。
体表は毛・羽・鱗・皮膚・甲殻の違いが分かるように描き分けます。
体色は実際の体色にもとづきます。雌雄差や季節差がある場合は、指定がなければ標準的な成体を描きます。

■ 画風
写真の複製ではなく、高品質な2Dの自然史イラストにします。
明快な輪郭、整理された色面、柔らかな多段階の陰影で立体を表現します。
毛を一本ずつ描くような高周波テクスチャに頼らず、大きな面と質感の差で描き分けます。
学校教材の挿絵より魅力があり、かつ生物学的に正しいこと。

■ 禁止
過度なデフォルメ。巨大な丸目。人間的な笑顔、眉、まつ毛。人間のような直立や身振り。
架空の角、ドリル、宝石、鎧、装備、発光、植物装飾。
王冠やエフェクトなど、希少さを画像へ焼き込む表現。
文字、ロゴ、UI、フレーム、キャプション、透かし、署名。
地面、台座、影、風景、比較線、説明パネル。
既存の有名IPキャラクターに似せること。
{speciesSpecificNegative}

■ 出力仕様（厳守）
1024×1024ピクセル、正方形、PNG。
本物の透明背景（アルファチャンネルを持ち、実際に完全透明な画素が存在すること）。
単色背景を残さない。透明を表す市松模様を描かない。足元に不透明な楕円影を置かない。
生きものは1体のみ。全身。中央配置。自然で識別しやすい3/4方向。
主要部位（頭・四肢・尾・角・翼など）を画面外で切らない。
種名・学名・特徴文は入れない（後工程でUIが配置する）。

最終出力は、上記を満たした画像1枚のみとします。
```

## 生成後

1. `npm run validate:encyclopedia` を実行し、終了コード0を確認する（`2`＝未生成を合格として扱わない）。
2. 目視で確認する：1体・全身か／対象種として識別できるか／文字・台座・風景が無いか／ファンタジー装飾が無いか。
3. 目視していない画像を「品質合格」と報告しない。
