# RARE — 実在が確認された希少形態

対象：アルビノ、白変種（リューシズム）、黒化個体（メラニズム）、斑個体、黄色変異など、**科学的に実在が確認されている**希少な表現型。共通方針は [00_ART_DIRECTION.md](00_ART_DIRECTION.md)。

保存先：`assets/encyclopedia/rare/<id>.png`

## RARE の定義（誤用を防ぐ）

- **単に珍しい動物・見つけにくい動物をRAREにしない。**
- **絶滅危惧種であることだけを理由にRAREにしない。** 保全状況はレアリティと無関係の別情報。
- **アルビノと白変種を混同しない。** アルビノはメラニンを全く作れず眼が赤い。白変種は色素が部分的に欠け、眼の色素は保たれる。
- 存在が確認できない形態を創作しない。
- **必ず通常種へ関連付ける**（`baseSpeciesId` 必須）。骨格・体型・脚数・体表構造は通常種と同一。

## 生成後

1. `npm run validate:encyclopedia` を実行し、終了コード0を確認する。
2. **通常種の画像と並べて骨格が一致しているか**を目視で確認する。骨格が変わっていたら不合格。
3. 指定された変異分類（アルビノ／白変等）と眼の色・模様の有無が整合しているか確認する。
