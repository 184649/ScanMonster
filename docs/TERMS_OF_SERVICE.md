# WORLDAWN 利用規約

最終改定日: 2026-07-10

本規約は、WORLDAWN（以下「本アプリ」）の利用条件を定めるものです。本アプリを利用した時点で、本規約に同意したものとみなします。

> 本規約は運営者が本アプリの仕様に基づき作成した初版であり、弁護士等の専門家によるレビューを受けたものではありません。正式公開にあたっては専門家の確認を推奨します。

## 1. サービス内容
本アプリは、利用者が身のまわりのバーコードやQRコードをスキャンしてキャラクターを発見し、キャラクターごとの公式発見番号・発見証明・図鑑・称号などを収集する娯楽サービスです。

## 2. 利用資格
本アプリは年齢を問わず利用できますが、未成年の方は保護者の同意のうえで利用してください。ログインは不要で、初回起動時に匿名アカウントが自動的に作成されます。

## 3. 禁止事項
利用者は次の行為をしてはなりません。
- サーバー・APIへの不正アクセス、脆弱性の悪用
- 自動化ツール・ボット・スクリプトによる大量アクセスやスキャンの自動化
- 公式発見番号の不正取得（number farming）、番号の売買・詐取を目的とした行為
- 不正な手段による複数アカウントの大量作成・悪用
- フレンドQRの改ざん・不正利用、他者になりすます行為
- 本アプリの運営を妨げる行為、法令・公序良俗に反する行為

## 4. 公式発見番号の性質
公式発見番号は本アプリ内の記録であり、**金銭的価値・所有権・財産的権利を保証するものではありません**。番号や発見証明の現実世界での売買・譲渡・換金を運営は認めません。運営はサービス運営上必要な範囲で番号体系・記録を調整することがあります。
