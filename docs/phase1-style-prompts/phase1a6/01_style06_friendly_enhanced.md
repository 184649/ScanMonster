> **本案の担当領域**：STYLE 06 の「顔つきが厳しい／野性味が強い」問題を、**顔・目・配色コントラスト・頭部比率**の調整で解決する。羊毛量と角の造形は控えめな追加にとどめ、02（羊らしさ強化型）と役割を重複させない。

## 維持するもの

- 角内側の特徴的な負の空間（**遠目でも読めること**）
- 頬から胸へ斜めにつながる一本の明快な羊毛ライン
- 現在の STYLE 06 に近い全体シルエット
- 堂々とした立ち姿
- normal としての明快さ

## 変更するもの

| 部位 | 調整内容 |
|---|---|
| 目 | 細すぎない自然な形。目尻の強い吊り上がりを弱める |
| まぶた | 柔らかくする |
| 顔 | 少しだけ短く、幅を持たせる |
| 鼻先 | ヤギのように細くしない |
| 口元 | 穏やかに。ただし人間の笑顔にしない |
| 配色 | 顔と脚の濃色コントラストを弱める |
| 頭部 | 現実より **約20〜25%大きく** |
| 脚 | 現実より **約10%だけ短く**（短足にしない） |
| 羊毛 | 現在より**少し**増やす。頬・胸・腰へ柔らかな房を追加 |
| 角 | 現在より少し丸みのある面へ整理 |
