> **本案の担当領域**：STYLE 06 の「野生羊・ムフロン・ヤギ寄りに見える」「羊毛量が少なく滑らかな体表に見える」問題を、**種の同定性（頭部幅・鼻先・耳・瞳孔）と羊毛量・角の硬質感**の調整で解決する。01（親しみ強化型）が顔の表情と親しみを担当するのに対し、本案は**家畜羊としての正しさ**を担当する。

## 維持するもの

- 角内側の特徴的な負の空間（**遠目でも読めること**）
- 頬から胸へ斜めにつながる一本の明快な羊毛ライン
- STYLE 06 固有の記憶しやすい外周
- 写実へ戻らない大きな色面
- normal としての明快さ

## 変更するもの

| 部位 | 調整内容 |
|---|---|
| 頭部 | 家畜羊らしい**幅のある**頭部へ |
| 鼻先 | ヤギ寄りの細長い鼻先を避ける |
| 瞳孔 | 横長瞳孔を**明確に** |
| 耳 | 家畜羊らしい自然な**横向き**へ |
| 配色 | 顔と脚の濃色部分を少し明るく |
| 羊毛量 | 首・胸・胴・腰で**増やす** |
| 羊毛の質 | 細かな巻き毛ではなく**大きな柔らかな房** |
| 羊毛の連続 | 胸から胴へ厚みを連続させる |
| 胴体 | 横長だが細長くしすぎない |
| 脚 | 自然な太さを持たせる |
| 角 | 野生羊の硬い骨格感を弱め、**滑らかな2〜4面**へ整理 |
